# generator-ss-gulp

yeoman generator to install gulp

## Requirements

* Node.js
* yarn
* yo

## Install

```bash
$ git clone https://github.com/suzuki-shunsuke/generator-ss-gulp
$ cd generator-ss-gulp  
$ npm link
```

Then generate your new project:

```bash
$ yo ss-gulp
```
